## 1.一阶滤波

### 算法简介/定义

一阶滤波器是一种基本的滤波器，用于平滑信号或减少噪声。常见的一阶滤波器包括低通滤波器和高通滤波器。以下是关于一阶滤波器算法的介绍。

#### 算法特点

##### 一阶低通滤波

一阶低通滤波器允许低频信号通过，同时衰减高频信号。其离散时间实现可以通过以下公式表示：

$$
{y[n]=\alpha x[n]+(1-\alpha)y[n-1]}
$$
